To run locally, do the usual:
1. Create a Python 3.6 virtualenv

2. Install dependencies::

    pip install -r requirement.txt

3. Migrate Code:
    ./manage.py migrate

4. create inital data
    python3 create_data.py

5. Run the Server:
    ./manage.py runserver
